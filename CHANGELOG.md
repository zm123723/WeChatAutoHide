# 更新日志

本项目的所有重要更改都将记录在此文件中。

## [1.0.0] - 2024-01-XX

### 新增
- ✨ 监控微信消息列表功能
- ✨ 自动隐藏指定联系人聊天
- ✨ 分组管理联系人（工作/生活/其他）
- ✨ 隐藏次数统计
- ✨ Material Design 3 界面
- ✨ 数据本地存储
- ✨ 无障碍服务状态检测

### 技术特性
- 🔧 使用 Room 数据库
- 🔧 Kotlin Coroutines 异步处理
- 🔧 ViewModel + LiveData 架构
- 🔧 AccessibilityService 实现自动化

### 已知限制
- ⚠️ 仅支持 Android 7.0+
- ⚠️ 需要提前开启消息免打扰
- ⚠️ 部分微信版本可能不兼容
